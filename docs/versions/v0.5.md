# v0.5 — Visual Overhaul

> Goal: make the game look and feel like a real TCG (Hearthstone, MTG Arena, TFT) using only CSS/SVG — no images, no canvas, no heavy libraries.

---

## What Changed

### Foundation
- Google Fonts: **Cinzel** (fantasy serif for headings/card names) + **Inter** (clean sans for stats/UI)
- `src/ui/theme.ts` — single source of truth for card themes, keyword colors, board tokens, font constants
- CSS keyframe animations: `cardGlow`, `shimmer`, `summonFlash`, `energyPulse`, `tauntPulse`, `shieldShimmer`, `slideIn`, `fadeInOut`, `particleFloat`, `hpFlash`

### Hand Cards — Collectible Card Design
- Cards enlarged to ~110px wide with layered layout: cost gem, art panel, name banner, keyword badges, stat orbs
- **SVG art** per template: shield (Vanguard), lightning (Striker), crystal (Glass Cannon), castle (Sentinel), flame (Phantom), fortress wall (Bulwark), crossed spears (Lancer)
- Cost gem: radial gradient with specular highlight
- Stat orbs: ATK (red, bottom-left) and HP (green, bottom-right) like Hearthstone
- Keyword badges: styled capsules with colored borders
- Fan layout: cards arc based on position (rotation + Y offset)
- Hover: lift (`translateY(-10px)`), scale, diagonal shimmer sweep
- Selected: pulsing glow animation

### Board Atmosphere
- Atmospheric radial gradients (blue top, warm bottom) + subtle grid texture overlay
- Player territories: blue-tinted rows for Player B, warm-tinted for Player A
- Center divider: glowing gradient line + "BATTLEFIELD" label in Cinzel
- Empty lanes: inset shadows (carved slot look)
- Active player territory: subtle inset gold glow

### Board Units — Mini Cards
- Mini SVG art (shared `CardArt` component at 30px)
- Template gradient background matching hand card theme
- Mini stat orbs (ATK/HP, 18px) at bottom
- Shield: shimmer animation overlay
- Spent: dimmed with red tint overlay
- Taunt target: pulsing amber border (CSS animation)

### Player HUD
- Circular avatar: crown SVG for active player, letter for inactive
- HP bar: 8px tall, gradient fill, dark border, HP number in Cinzel with glow
- Energy: diamond-shaped gems (rotated squares) with radial gradients + counter

### Controls & UI
- Buttons: Cinzel font, gradient fills, bottom-border depth trick (press effect), glow on hover
- Win screen: "VICTORY" text in Cinzel with gold glow/text-shadow
- Log: Inter font, left accent bar per message type (color-coded), slide-in animation
- Header: "TCG BLUEPRINT" with gold gradient text, styled turn/phase badges

### Animations & Polish
- Turn transition banner: "PLAYER X'S TURN" fades in/out on turn change
- Ambient particles: 6 CSS-animated dots drifting upward
- Card shimmer: diagonal highlight sweep on hover

## Constraints
- **No engine changes** — all 45 smoke tests pass unmodified
- CSS/SVG only — no images, canvas, or external UI libraries
- No new game mechanics or card types
- Designed for 1280px+ screens

---

## Out of Scope
- Sound effects
- Card play/summon animations (beyond CSS transitions)
- Mobile/responsive layout
- Card back design
- Drag-and-drop interactions
